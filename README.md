# Freshman_project Basics

Idea: App de restaurante tipo delivery

Objetivo final: Crear dos apps y poder conectarlas, una app para los clientes, otra para los del restaurante

Tools:
    - React Native
    - Js
    - Styles
    - Maybe Node.js for backend


First Objective:
Poder crear una base de la app, solamente la parte visual, aun sin mucho style, solo lo necesario para que se vaya viendo decente y con forma

Second Objective:
Meterle ganas al style, hacerla ver bonita y con imagenes, agregar colores, sombras, etc. (UI)

Thrid Objective:
Enfocarnos en la parte completamente funcional y experiencia del cliente (UX)

Four Objective:
Backend, base de datos, en algun lado hay q almacenar los login con sus contrasenas y usuarios, ademas de los carritos


# Puntos a tener en cuenta

    - Hay que dedicarle tiempo y amor (probably + than 8h per week)
    - Talvez solo alcance a hacer una sola app, pero le vamos a meter ganas
    - No estresarse (mayor enemigo)
    - Estrategia de crear y probar constantemente, no queremos que ya terminando encontremos errores muy grandes.
